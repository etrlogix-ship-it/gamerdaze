# 04 🌳 Terraria Server

**Category:** 🖥️ Game Servers
**Difficulty:** ⭐⭐ Intermediate
**Time:** 30-45 minutes

## What You'll Make
A dedicated 🌳 Terraria server. Your world, your rules, always online even when you're not.

## Requirements
- Tool: TShock
- Port: 7777 (TCP)
- Notes: Download TShock from tshock.org

## How to Run
```bash
cd code
python3 server_launcher.py
```

## Key Tips
- Keep server software up to date for security and bug fixes
- Back up your save files regularly (see project 17!)
- Set up auto-restart on crash (see project 19!)
- Monitor performance (see project 16!)

## Port Forwarding
Forward port **7777 (TCP)** on your router to your server's local IP.
See project 56 for a full port forwarding guide.
