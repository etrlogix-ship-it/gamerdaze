# 17 💾 Auto Backup System

**Category:** 🔧 Server Admin
**Difficulty:** ⭐⭐
**Time:** 25 minutes

## What You'll Make
Automatically back up your game world files on a schedule. Never lose your 500-hour survival world to a corrupt save or accidental deletion again.

## What You'll Learn
- File copying and compression
- Scheduled tasks with Python
- Backup rotation (keeping last N backups)

## What You'll Need
- Python 3 (standard library only)

## How to Run
```bash
cd code
python3 auto_backup.py
```
