# 15 🚀 Multi-Server Launcher

**Category:** 🖥️ Game Servers
**Difficulty:** ⭐⭐ Intermediate
**Time:** 20 minutes

## What You'll Make
A single control panel to launch, stop, and check the status of multiple game servers from one terminal menu. No more hunting through folders!

## What You'll Learn
- Managing multiple processes in Python
- Building interactive terminal menus
- Process monitoring

## How to Run
```bash
cd code
python3 multi_launcher.py
```

## Features
- Launch any configured server with a single keypress
- See which servers are running and their uptime
- Stop individual servers or all at once
- Add your own servers to the config list
