# 18 📋 Player Whitelist Manager

**Category:** 🔧 Server Admin
**Difficulty:** ⭐
**Time:** 15 minutes

## What You'll Make
A friendly interface to manage who can join your server. Add, remove, and search players without editing JSON files by hand.

## What You'll Learn
- JSON file handling
- Command-line tools
- Server administration

## What You'll Need
- Python 3 (standard library only)

## How to Run
```bash
cd code
python3 whitelist_manager.py
```
