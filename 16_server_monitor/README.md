# 16 📊 Server Performance Monitor

**Category:** 🔧 Server Admin
**Difficulty:** ⭐⭐
**Time:** 20 minutes

## What You'll Make
Watch your server's CPU, RAM, disk, and network usage in real time from a live terminal dashboard. Know when your server is struggling before players start complaining.

## What You'll Learn
- Real-time system monitoring
- Terminal dashboards with curses
- CPU/RAM/disk metrics

## What You'll Need
- psutil library: pip install psutil

## How to Run
```bash
cd code
python3 server_monitor.py
```
