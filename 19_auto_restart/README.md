# 19 🔄 Auto-Restart Watchdog

**Category:** 🔧 Server Admin
**Difficulty:** ⭐⭐
**Time:** 20 minutes

## What You'll Make
Monitors your game server and automatically restarts it if it crashes. No more waking up to find the server went down at 3am and nobody noticed.

## What You'll Learn
- Process monitoring
- Watchdog pattern
- Automatic recovery

## What You'll Need
- Python 3 (standard library only)

## How to Run
```bash
cd code
python3 watchdog.py
```
